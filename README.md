# mbl-managemet-events

## Deployment

Rodando projeto via docker-compose

```bash
  docker-compose up
```
vai subi imagem com postgress e api


### Executando local
Node >=16

```bash
    npm install
    npm run dev
```

**Obs**: necessario postgres rodando na porta padrão!

## Documentação

no endpoint /docs da acesso a api swagger.
```bash
    localhost:3000/docs
```
